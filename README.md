# E-Ticaret Uygulaması
Html5, Css3, Bootstrap, C#, OOP, ASP.Net MVC, Entity Framework, MSSQL, LINQ, CodeFirst.

Proje ASP:NET MVC'de hazırlanmış bir e-ticaret sitesidir. Bir e-ticaret siztesinde olması gereken temel özellikler projede mevcuttur. Codefirst kullanılmıştır vveritabanını aktifleştirmeniz için web.config dosyasındaki ayarlamaları yapmanız gerekmektedir. Asp.Net Identity Frameworku ve Owin kullanılmıştır. Veritabanı Inıtıalize edilirken kullanıcı adı ahmetsokmen şifre 123456 olarak admin hesabı oluşturulur detaylar ilgili classın kodlarında mevcuttur.
